

# Day 12: Smart AI System (OpenAI + Data)

## 📌 Description
This project is a hybrid AI system that combines OpenAI API with structured data (CSV).
It can answer both general questions and data-related queries.

## 🚀 Features
- Uses OpenAI API for intelligent responses
- Reads data from CSV file
- Answers questions like average and highest marks
- Hybrid system (AI + Data)

## 🛠️ Technologies Used
- Python
- OpenAI API
- CSV module

## 📂 Files
- day12.py → Main chatbot system
- students.csv → Dataset

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Set API key:
   export OPENAI_API_KEY="your_api_key"

3. Run the project:
   python3 day12.py

## 💡 Example Inputs
- average marks
- highest marks
- what is AI?
- how to study effectively?
